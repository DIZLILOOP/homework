#Конфигурационный парсер на Python

Простой парсер для учебного конфигурационного языка → XML

##Файлы проекта
- `config_parser.py` - основной парсер
- `test.py` - тесты для проверки
- `example1.config` - пример конфигурации приложения
- `example2.config` - пример конфигурации БД
- `example3.config` - пример сетевой конфигурации

#Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/OIZLILOOP/homework.git
cd homework

# Запустить тесты
python test.py

# Использовать парсер
python config_parser.py example1.config
